Add Post Thumbnails to the WordPress Post and Page Management Screens
== Description ==
This plugin adds a post thumbnails to the WordPress Post and Page Management Screens. 
The idea of this plugin comes from Sarah Gooding..
Copyright http://wordpress.stackexchange.com/questions/1567/best-collection-of-code-for-your-functions-php-file/6021

== Installation ==
1. Upload `add-thumbnails-to-post-list/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
